# Project SEO Rules

To maintain consistency and high SEO performance for IZODIAMANT, follow these rules:

## 1. Page Titles (Meta Titles)
- **Length Constraint:** Keep meta titles under 580 pixels (roughly 55-60 characters).
- **Format:** `[Short Descriptive Title] | IZODIAMANT`
- **Reference Pages:** For reference pages, avoid long titles. Use "Sanace [type] zdiva, [Location]" instead of "Kompletní sanace vlhkého cihelného zdiva rodinného domu, [Location]".

## 2. Mandatory Phrases
- **Brand Promise:** Always include the phrase **"Vracíme zdraví vaší stavbě."** (including the dot) in:
    - Main site description (`layout.tsx`)
    - Hero section description
    - Footer description
    - Reference page meta descriptions
- **Forbidden Phrases:** NEVER use **"s doživotní zárukou"**.

## 3. Metadata Structure
- **Reference Pages:** Meta descriptions should follow the pattern: `Sanace zdiva: [Title]. [Location]. Vracíme zdraví vaší stavbě.`
- **OpenGraph:** Ensure OpenGraph titles and descriptions match the meta tags for consistency.

## 4. Visual Assets
- All images should have descriptive `alt` tags related to masonry remediation and the specific technology used.
